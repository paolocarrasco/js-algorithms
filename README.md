-------

    1. Insertion sort
    1. Selection sort
    1. Merge sort
1. Known algorithms
    1. Hanoi towers

Requirements
------------

In order to run the code, it is needed just NodeJS > 4.

How to run them
---------------

For each file execute:

```
node <path to file>
```

Example, to run the bubble sort:

```
node sorting/bubble-sort.js
```
